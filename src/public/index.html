Ludo Game Web App

This project is a ES6 implementation of the popular board game 'Ludo'. It uses module-like programming implemented via Classes, with a sequence call architecture shown on the github page. It uses the native Canvas API offered by the Mozilla Developer Network with a sequential call for the animation instead of the animation loop offered by the API. Additionally, it uses Asynchronous programming by handling promises given by functions that require an un-sequential (asynchronous) call.
